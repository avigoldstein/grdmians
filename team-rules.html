Behavior

  • Communication: Slack
  • If there is no prior notification, we will wait ten minutes until we start a meeting without a member
  • If somebody does not respond on Slack in a timely manner, we will resort to messaging on Facebook and phone calls in order to get in touch
  • We expect a 26 hour notice if somebody will not meet a deadline
  • We should respond to emails within 26 hours
  • If one is more than 20 minutes late to a meeting without a valid reason they owe the whole team ice cream

Coding Practice

  • Never edit production branch directly
  • Every developer has their own branch
  • Message dev channel on Slack after committing
  • Git workflow
    • Checkout production branch, then `git pull`
    • Checkout local branch, then `git merge prodbranch`
    • Commit - `git commit -a -m ‘commit message’`
    • Checkout prod branch, then `git merge localbranch`
    • Push - `git push`

User Types and Use Cases

First Time User

  • Create public key
  • Publish key

Encrypted Message with New Person

  • Add person public key
  • Create a pub/priv key
  • Share key via their public key
  • Enable the shared key

Encrypted Message with Returning Users

  • Enable shared key

Encrypted Message in Group

  • Create a pub/priv key
  • Share with all users via their pub key
  • Enable shared key

Use Cases

Send contact request

  • Copy own ID
  • Send over unsecure channel

Accept contact request

  • Click button
  • Choose pet name or accept default pet name

Create new group

  • Open Grd.Me popup
  • Click "Create Group"
  • Search and select contacts
  • Create group

Added to a group

  • Push notification telling you're added to group
  • Adds members in the group not currently in contact list

Send message via native text box

  • Type message in-browser
  • ⌘/ctrl + E
  • Send

Send message via Grd.Me secure popup text box

  • ⌘/ctrl + ⇧ + E
  • Type message
  • Click “Encrypt” or press ⌘/ctrl + E
  • Copy and paste
  • Send

Update contact name

  • Search contacts
  • Click desired contact
  • Click "Edit Name"

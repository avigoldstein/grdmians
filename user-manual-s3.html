User Manual and Administrator Manual

General Information

The purpose of Grd Me is to have an easy, accessible way to encrypt text and use everyday communication channels to send messages in a secure way. You can use any messaging service to send encrypted messages to those who you have specifically shared your ID with, so that only they can see the decrypted text that you sent.

System Overview

Currently, Grd Me is compatible with Google Chrome, and the extension is usable on a desktop only. Because the extension is still in its development stage, in order to use the extension, a user must download the code from the Github repo and run the file from the Chrome "developer mode."

Getting Started

To get started with Grd Me, follow these steps:

Setting up

  1. Enable the Grd Me extension on Chrome
  2. Open the extension by clicking on the icon to trigger the popup box
  3. Register a name, which will be associated with you personally
  4. A unique ID will then be generated and stored in the key server, and this will be a one-time occurrence

Adding contacts

  1. Have your intended contact send you their key by clicking the blue "Copy My Key" button in the extension popup
  2. Grd Me will recognize the key as a Grd Me key, and prompt you to add them as a contact
  3. If you are not online when a user sends you their key, they will show up as a pending contact in your list when you open your extension
  4. Once you've added them, you can either keep the name that they set, or you can change it by clicking the edit button

Messaging users

Once you've populated your contacts, you can now message those users using any messaging service encrypted messages, and Grd Me will automatically decrypt the messages for them. To do this, follow these steps:

  1. In a text field on a messaging service (such as Facebook messenger), hit Ctrl/Command+e on your keyboard
  2. A textbox will pop up, and you can then type your message
  3. Press "encrypt and submit" when you have finished composing the message
  4. Your encrypted message is now copied to your clipboard - you can send it by pasting into the text field that you would like to use to send your message to

Enabling Multi-Device

In order to enable using GrdMe across multiple devices, you must sync your GrdMe information through Google Chrome

  1. Open up settings for Google Chrome, and click on the Advanced Sync Settings button. You can either sync everything, or you can select what to sync (in this case be sure to have Extensions selected)
  2. when opening Chrome on another computer, make sure that you are signed in with the Chrome account (synonymous with a gmail account) you were signed into when originally setting up your GrdMe
  3. From there, all of your information should sync
